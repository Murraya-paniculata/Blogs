max-height应用在展开收起效果

2018.10.24 tom  热度 ℃

前端css属性中max-height我们使用的并不多，今天给大家分享一个max-height属性实现展开收起效果。

效果图如下：

话不多说直接上代码。

“展开收起”无疑是我们最常见的前端效果之一，一般的做法是采用js控制元素的display属性。虽然可以实现，但是希望元素展开收起时能有明显的高度滑动效果。传统实现可以使用 jQuery 的 slideUp()/ slideDown()方法，但是，在移动端，因为 CSS3 动画支持良好，所以移动端的 JavaScript 框架都是没有动画模块的。此时，使用 CSS 实现动画就成了最佳的技术选型。 我们的第一反应就是使用 height + overflow:hidden 实现。但是，很多时候，我们展开的元素内容是动态的，换句话说高度是不固定的，因此，height使用的值是默认的 auto，应该都知道的 auto 是个关键字值，并非数值，正如 height:100%的 100%无法和 auto相计算一样，从 0px 到 auto 也是无法计算的，因此无法形成过渡或动画效果。

但是如果使用“max-height”属性的话，就不会有这个问题，我们只需要展开时把max-height设置为一个足够大的值。当展开的dom高度小于这个max-height时，元素高度就是内容撑开的高度。

缺点：

​ 如果将max-height设置的太大的话，动画会产生一定的延迟，但是如果我们取一个合适的max-height值得话，用户几乎察觉不到。
